# Portfolio website

## Tech stack

- Nuxt 3
- Vue 3 Composition API
- Nuxt UI
- Tailwind
- TypeScript
- Prettier (& prettier-plugin-tailwindcss) enforced
- ESLint enforced
- Firebase hosting

## Third party libraries

- emailjs for the contact form
- Google reCAPTCHA
- Google Appointment Schedule
- Nuxt SEO
- Nuxt Security
